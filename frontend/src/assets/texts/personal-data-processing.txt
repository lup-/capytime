1. Общие положения

1.1. Настоящие Правила определяют порядок обработки персональных данных пользователей сервиса CapyTime в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

1.2. Оператор осуществляет обработку персональных данных на законной и справедливой основе.

2. Принципы обработки персональных данных

2.1. Обработка персональных данных осуществляется с соблюдением следующих принципов:
- законность целей и способов обработки;
- соответствие целей обработки заявленным;
- недопустимость объединения баз данных, созданных для несовместимых целей;
- обработка только тех данных, которые необходимы для заявленных целей;
- соответствие объёма и характера данных заявленным целям;
- точность, достаточность и актуальность данных;
- хранение в форме, позволяющей определить субъекта, не дольше чем требуют цели.

3. Цели обработки персональных данных

3.1. Персональные данные обрабатываются в следующих целях:
- регистрация и идентификация специалистов;
- предоставление услуг по записи на приём;
- отображение публичной информации о специалистах;
- коммуникация с пользователем;
- исполнение обязательств по договору.

4. Объём обрабатываемых данных

4.1. Оператор обрабатывает следующие категории персональных данных специалистов (публичные данные, разрешённые к публикации):
- имя и фамилия;
- фотография (аватар);
- специализация и направления работы;
- форматы работы (онлайн/очно);
- адрес для очных встреч (при использовании);
- ссылка на видеоконференцию (при использовании).

4.2. Оператор обрабатывает следующие данные клиентов:
- имя;
- контактные данные (телефон, email);
- информация о формате и времени записи.

4.3. CapyTime не хранит данные о расписании и записях — эта информация хранится в подключённых сторонних сервисах (календарях).

5. Правовые основания обработки

5.1. Обработка персональных данных осуществляется на основании:
- согласия субъекта персональных данных;
- исполнения договора, стороной которого является субъект;
- требований законодательства.

6. Обязанности оператора

6.1. Оператор обязан:
- обеспечить защиту персональных данных от несанкционированного доступа;
- немедленно уведомлять субъекта о выявленных нарушениях;
- прекратить обработку данных по требованию субъекта.

7. Ответственность

7.1. Оператор несёт ответственность за соблюдение требований законодательства о персональных данных.
