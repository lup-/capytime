1. Общие положения

1.1. Настоящая Политика конфиденциальности (далее — Политика) определяет порядок
сбора, обработки, хранения и защиты персональных данных пользователей сервиса
CapyTime (далее — Сервис).

1.2. Оператором персональных данных является ООО "Тауистема"
(ОГРН: 5077746441691, ИНН: 7707622619) (далее — Оператор).

1.3. Обработка персональных данных осуществляется только с согласия пользователя,
выраженного путем проставления соответствующей отметки (чекбокса) при
регистрации, авторизации или отправке данных через формы Сервиса.

1.4. Правовыми основаниями обработки персональных данных являются:
Гражданский кодекс РФ, Пользовательское соглашение (оферта) Сервиса,
а также согласие пользователя на обработку персональных данных.

1.5. Оператор осуществляет следующие действия с персональными данными: сбор, запись,
систематизация, накопление, хранение, уточнение (обновление, изменение),
извлечение, использование, передача (предоставление, доступ), блокирование,
удаление, уничтожение.

2. Категории субъектов и собираемые данные

Сервис обрабатывает данные двух категорий пользователей: Психологов
(предоставляющих услуги) и Клиентов (записывающихся на приём).

2.1. Данные Психологов:

  - имя, фамилия, адрес электронной почты;
  - данные, получаемые через API сервиса календарей (список календарей и события
    в них, токены доступа — access token, refresh token — для выполнения
    операций от имени пользователя);
  - фотография профиля;
  - список проблем, с которыми работает Психолог;
  - направления/специализация;
  - опыт работы (лет);
  - цена онлайн-консультации;
  - цена очной консультации;
  - адрес для очных приёмов;
  - информация "Обо мне";
  - данные об образовании;
  - данные о профессиональном опыте;
  - ссылка на персональную видеоконференцию.

2.2. Данные Клиентов:

  - имя, фамилия (по желанию);
  - номер телефона и/или адрес электронной почты (по желанию);
  - дата и время записи на приём.

2.3. Технические данные (файлы cookie):

  - IP-адрес, тип браузера, данные об устройстве, файлы cookie. Эти данные
    собираются автоматически для обеспечения корректной работы Сервиса и
    базовой аналитики. Пользователь может отключить использование файлов
    cookie в настройках своего браузера.

2.4. Оператор не собирает и не обрабатывает
специальные категории персональных данных (касающиеся состояния здоровья,
диагнозов, интимной жизни) и биометрические данные. Клиентам строго запрещается
указывать медицинскую информацию или описывать свое психологическое состояние в
любых свободных текстовых полях Сервиса.

3. Цели обработки данных

3.1. Данные используются строго в рамках функциональности Сервиса — организации
записи Клиентов на приём к Психологу. Никакие данные не используются для
таргетированной рекламы, профилирования или иных целей, не связанных с
основной функцией Сервиса.

3.2. Данные Психологов (включая данные из Календаря) используются для:

  - создания отдельного календаря "Записи CapyTime";
  - проверки доступности: чтение занятых временных слотов для предотвращения
    двойных записей;
  - создания, изменения и чтения событий (записей Клиентов) с указанием времени
    и ссылки на видеоконференцию;
  - идентификации учётной записи.

3.3. Данные Клиентов используются для:

  - оформления записи к Психологу;
  - отправки сервисных уведомлений (подтверждение записи, перенос, отмена,
    ссылка на видеоконференцию).

4. Передача данных третьим лицам и трансграничная передача

4.1. Оператор не продает данные пользователей. Передача (или поручение
обработки) данных третьим лицам осуществляется исключительно в целях
обеспечения технической работы Сервиса.

4.2. Для работы Сервиса данные могут передаваться следующим категориям третьих лиц:

  - Сервисы календарей (выбранные Психологом).
  - Почтовые сервисы (для отправки уведомлений о записях, передаются только имя
    Клиента, дата, время и ссылка на видеоконференцию).

4.3. Трансграничная передача:
В случае использования Психологом иностранных сервисов календарей, Оператор
осуществляет трансграничную передачу данных на территорию иностранных государств.
Такая передача осуществляется в соответствии с требованиями ст. 12
ФЗ «О персональных данных» для исполнения договора (оферты) с пользователем.

5. Хранение и защита данных

5.1. Первоначальный сбор и обновление баз данных граждан Российской Федерации
осуществляется с использованием баз данных, находящихся на территории
Российской Федерации.

5.2. Для защиты данных применяются:

  - шифрование соединения (TLS) при передаче данных;
  - доступ к данным предоставляется только авторизованным серверным компонентам;
  - токены доступа хранятся в защищённой базе данных только на сервере (не
    передаются в браузер клиента) и автоматически обновляются. Без
    обновления токены доступа необратимо устаревают.

6. Сроки хранения и удаление данных

6.1. Персональные данные хранятся до достижения целей обработки или до момента
отзыва согласия / удаления учётной записи пользователем.

6.2. Пользователь может в любой момент:

  - отключить интеграцию с сервисом календарей в настройках профиля;
  - запросить полное удаление своих данных, обратившись по адресу
    support@capytime.ru.

6.3. При отключении сервиса календарей токены доступа и код календаря немедленно
удаляются из базы данных Оператора.

6.4. При удалении учётной записи (или получении отзыва согласия) все данные
пользователя, включая токены и историю записей, безвозвратно уничтожаются
Оператором в течение 30 дней, если иное не предусмотрено законодательством
РФ.
